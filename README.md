# Markdown Viewer

An offline-first Next.js PWA for editing Markdown files and keeping local notes. It supports direct file-system access in Chromium browsers, live preview, local persistence, install prompts, and a custom service worker.

## Scripts

```bash
pnpm dev
pnpm build
pnpm start
pnpm lint
```

Open `http://localhost:3000` during development.

## Routes

- `/login`
- `/portal/markdown`
- `/portal/notes`

## Project Structure

```text
backend/                  Backend services, Supabase client, DB migrations, server handlers
src/app/                  Next.js App Router routes, layout, thin API route adapters, metadata
src/features/markdown/    Markdown workspace UI, file-system helpers, markdown parsing
src/features/notes/       Notes UI and note domain types/mappers
src/features/portal/      Portal shell, login, session-level app composition
src/features/pwa/         PWA install prompt, offline banner, service-worker update UI
src/types/                Global browser/API type declarations
public/                   Static assets, icons, and service worker
```

Cross-feature imports use the `@/*` alias. Backend imports use the `@backend/*` alias. Files inside the same feature use local relative imports.

## PWA

The web app manifest is generated by `src/app/manifest.ts`. The service worker lives at `public/sw.js`.

## Supabase Notes Backend

For first-time setup, run `backend/supabase/init.sql` in your Supabase SQL editor. It creates the `public.notes` table, defaults, indexes, update trigger, and reloads the Supabase schema cache.

If you use migrations, the same schema is available at `backend/supabase/migrations/001_create_notes.sql`.

To insert a default welcome note, set `SEED_OWNER_EMAIL` in `.env.local` and run:

```bash
pnpm db:seed
```

Add these environment variables locally and in your deployment:

```bash
SUPABASE_URL=https://your-project.supabase.co
SUPABASE_SERVICE_ROLE_KEY=your-service-role-key
```

The service role key is used only by Next.js API routes. Do not expose it with a `NEXT_PUBLIC_` prefix.
